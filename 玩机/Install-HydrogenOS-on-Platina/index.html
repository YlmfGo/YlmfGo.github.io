小米8青春版 氢OS刷入&体验

前言

尝试过许多类原生，总是在本地化上要折腾许久，所以想试试本地化相对完善的氢OS。去年的这个时候，我用了两个月rsyhan移植的氢OS，整体感觉不错。

但，rsyhan的包已经很难找到了，YouLinw的包还是Android 9……

想起国外的Healer还有移植氢和氧，就去翻了下，找到啦。

刷入

  1. 备份数据，进入Recovery，这里使用的是Orange Fox 11
  2. 清除Data、Cache、System、Vendor、内置存储
  3. 刷入底包
  4. 再刷入移植包
  5. 重启，Enjoy it!

开机后配置

  1. 移动数据无法使用，需要插入两张SIM卡，在 设置-SIM 卡与移动网络-默认流量卡 中选择卡2，再选择卡1即可使用。（此时卡2可拔出）
  2. 包自带的内核不是很好用，我推荐刷入Alpaca内核
  3. 包默认没有带中文输入法、浏览器、相册、计算器、相机，请自行准备安装包解决。相机请使用Google相机，其他按个人喜好安装。
  4. 移植者Healer往包内添加了基础的Google服务，可以直接使用Play Store
  5. 想要开启双击亮屏，请刷入Magisk模块。（开启后息屏通知主动显示将不可用）
  6. 推荐开启小横条沉浸，使用天伞桜的教程
  7. 氢OS深色模式切换藏的比较深，可使用深色模式切换来快速切换

体验

这个包我已经使用一周了，大bug没遇到过，整体使用还是十分稳定的。

指纹、人脸解锁可用

机型会被改成OnePlus 7，可以修改build.prop再改回来

后记

感谢Healer的移植！

有任何问题可以在酷安图文下留言。
